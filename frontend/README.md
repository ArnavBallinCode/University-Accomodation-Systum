# Frontend (React + TypeScript + Tailwind)

## Stack

- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Vite

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

Vite starts on `http://localhost:5173` and proxies API calls to backend `http://localhost:8000`.

## Production Build

```bash
npm run build
```

Build output is generated in `dist/` and served by the FastAPI backend at `GET /`.

## Preview Build

```bash
npm run preview
```
